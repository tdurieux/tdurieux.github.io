# SyncTeX-js

SyncTeX is a new method in TeXLive 2008 for enabling synchronization between source TeX files and the resulting PDF file. This repository contains a JavaScript implementation of for SyncTeX. It has been originally developed for \BlueLaTex , an open-source LaTeX editor for the Web. SyncTeX-js is now used in LaTeX Workshop as the main SyncTex parser for 330,905 installs.
